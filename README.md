# Myntra-Clone

Online Shopping Site for Fashion & Lifestyle in India. India's Fashion Expert brings you a variety of footwear, Clothing, Accessories and lifestyle products

# Tech - Stacks
- React
- Chakra - Ui
- MongoDB - Atlas
- Node js
- Express
- Redux

# Features
- Authentication
- Searching 
- Filtering products
- cart functionality
